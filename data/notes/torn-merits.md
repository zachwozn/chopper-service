- Merits (overview): merits are permanent character upgrades bought with merit points (earned from awards/achievements). Most can be bought multiple times, stacking their effect each level. They cover stats, combat, crime, money, and weapon proficiency. [source: Torn API (merits) | 2026-07-17]
- Merit - Nerve Bar: +1 to your maximum nerve per upgrade. [source: Torn API merits | 2026-07-17]
- Merit - Critical Hit Rate: +0.5% critical hit chance per upgrade. Crits are blows to an opponent's head, throat, or heart during an attack. [source: Torn API merits | 2026-07-17]
- Merit - Awareness: +20% to the number of items you find in the city (does not raise their quality). [source: Torn API merits | 2026-07-17]
- Merit - Masterful Looting: +5% to the money you mug from opponents. [source: Torn API merits | 2026-07-17]
- Merit - Stealth: +0.2 stealth level per upgrade, making you more likely to stay hidden (shown as 'Someone') when you beat an opponent so they don't know who hit them. [source: Torn API merits | 2026-07-17]
- Merit - Bank Interest: +5% to your investment bank interest (kicks in on your next investment). [source: Torn API merits | 2026-07-17]
- Merit - Hospitalizing: +5% to the hospital time you deal to opponents per upgrade, up to +50% fully upgraded. [source: Torn API merits | 2026-07-17]
- Merit - stat merits (Brawn / Sharpness / Evasion / Protection): each gives a passive +3% to a battle stat - Brawn = strength, Sharpness = speed, Evasion = dexterity, Protection = defense. These don't change your visible stat number but apply during attacks. [source: Torn API merits | 2026-07-17]
- Merit - Life Points: +5% to your maximum life. [source: Torn API merits | 2026-07-17]
- Merit - Crime XP: +3% passive Crime XP per upgrade, raising crime success rates. [source: Torn API merits | 2026-07-17]
- Merit - Education Length: -2% education course completion time per upgrade (applies to your next course). [source: Torn API merits | 2026-07-17]
- Merit - Addiction Mitigation: -2% to the negative effects of drug addiction per upgrade (less impact on battle stats, employee effectiveness, and education/gym access). [source: Torn API merits | 2026-07-17]
- Merit - Employee Effectiveness: +1 employee effectiveness per upgrade, earning more money for the company you work for. [source: Torn API merits | 2026-07-17]
- Merit - Weapon Mastery (mechanic): each weapon-type mastery gives +1% damage and +0.2 accuracy with that weapon type per upgrade. There are 11 mastery types (firearm and melee). [source: Torn API merits | 2026-07-17]
