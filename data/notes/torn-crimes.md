- Crimes 2.0 (overview): Torn's solo crime system (separate from faction Organized Crimes). There are 13 crimes spread across 6 categories: Vandalism, Theft, Counterfeiting, Fraud, Illicit services, and Cybercrime. Each crime has a specific enhancer item that boosts your success/results for that crime, and many crimes have a large pool of possible outcomes. [source: Torn API (crimes) | 2026-07-17]
- Crimes 2.0 - Theft crimes: Search for Cash (enhancer: Glasses), Shoplifting (enhancer: Mountain Bike), Pickpocketing (enhancer: Cut-Throat Razor), and Burglary (enhancer: Flashlight). [source: Torn API (crimes) | 2026-07-17]
- Crimes 2.0 - Fraud crimes: Card Skimming (enhancer: Duct Tape), Hustling (enhancer: Megaphone), and Scamming (enhancer: Ergonomic Keyboard). [source: Torn API (crimes) | 2026-07-17]
- Crimes 2.0 - Vandalism crimes: Graffiti (enhancer: Paint Mask) and Arson (enhancer: Windproof Lighter). [source: Torn API (crimes) | 2026-07-17]
- Crimes 2.0 - Counterfeiting crimes: Bootlegging (enhancer: High-Speed Drive) and Forgery (enhancer: Magnifying Glass). [source: Torn API (crimes) | 2026-07-17]
- Crimes 2.0 - other categories: Disposal is the Illicit Services crime (enhancer: Latex Gloves), and Cracking is the Cybercrime (enhancer: Office Chair). [source: Torn API (crimes) | 2026-07-17]
- Crime enhancers (mechanic): each crime has one matching enhancer item that improves it - e.g. Glasses for Search for Cash, Mountain Bike for Shoplifting, Cut-Throat Razor for Pickpocketing, Flashlight for Burglary, Windproof Lighter for Arson, Paint Mask for Graffiti, Office Chair for Cracking, Latex Gloves for Disposal.
