- Organized Crimes 2.0 (overview): OCs 2.0 are faction team heists (separate from solo Crimes 2.0). There are 31 scenarios across 5 faction tiers (spawn level 1-5, shown in-game as Greek letters lambda/sigma/phi/psi/omega), with difficulty ratings from 1 to 10. Each scenario needs a full team, one member per named role slot, and many roles require a specific item - some items are consumed on use, some are reusable. A few scenarios are locked behind completing a prerequisite OC first. [source: Torn API (organizedcrimes) | 2026-07-17]
- OC 2.0 tiers: the faction Organized Crimes upgrade branch unlocks each tier. Level 1 = lambda (difficulty 1-2), level 2 = sigma (difficulty 3-4), level 3 = phi (difficulty 5-6), level 4 = psi (difficulty 7-8), level 5 = omega (difficulty 8-10). Higher tiers pay more (scope return rises with difficulty). [source: Torn API (organizedcrimes) | 2026-07-17]
- OC 2.0 roles (all): Arsonist, Assassin, Bomber, Car Thief, Cat Burglar, Cleaner, Decoy, Driver, Enforcer, Engineer, Hacker, Hijacker, Hustler, Imitator, Interrogator, Kidnapper, Lookout, Looter, Muscle, Negotiator, Picklock, Pickpocket, Reviver, Robber, Saboteur, Smuggler, Sniper, Techie, Thief. Each is a slot a faction member fills; many need an item in hand. [source: Torn API (organizedcrimes) | 2026-07-17]
