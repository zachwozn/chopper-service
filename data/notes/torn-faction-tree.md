- Faction upgrade tree (overview): factions spend respect on a branching upgrade tree split into 9 groups - Core, Criminality, Fortitude, Voracity, Toleration, Excursion, Steadfast, Aggression, and Suppression. Most branches have many levels; higher levels often need a challenge (a faction-wide stat milestone) plus a respect cost. Below is what each branch does and its max effect. [source: Torn API (faction tree) | 2026-07-17]
- Faction tree - Core (armories & unlocks): unlock armory tabs for Weapons, Armor, Temporary weapons, Medical, Booster, Drug, Consumable, and Utility items; Point storage (donate/use points); Laboratory (unlocks the dirty bomb lab, needs 25 members). [source: Torn API (faction tree) | 2026-07-17]
- Faction tree - Core (Chaining): Chaining I-XIII raises your faction's max chain, from 10 all the way up to 100,000, each level gated behind reaching the previous chain milestone. [source: Torn API (faction tree) | 2026-07-17]
- Faction tree - Core (Capacity, Territory, OCs): Capacity I-X raises faction member capacity up to 100; Territory I-XXX lets a faction hold up to 30 territories; Organized Crimes I-V unlock progressively bigger OC tiers (Introductory Tasks up to Elaborate Campaigns), gated behind completing OC scenarios. [source: Torn API (faction tree) | 2026-07-17]
- Faction tree - Criminality: Nerve I-XL raises each member's max nerve by up to +40 (gated behind total offences committed); Crimes I-X gives up to +10% crime exp & skill gain; Jail time I-XV cuts jail time up to -30%; Bust nerve reduces bust nerve cost by up to 3; Bust skill I-X raises bust success chance up to +50%. [source: Torn API (faction tree) | 2026-07-17]
- Faction tree - Fortitude (survival/healing): Medical cooldown adds up to +180 min max medical cooldown; Reviving I-X cuts revive energy cost from 75 down to 25; Hospital time I-XXV reduces hospital time up to -25%; Life regeneration adds up to +4% life per tick; Medical effectiveness adds up to +30% medical item effectiveness. [source: Torn API (faction tree) | 2026-07-17]
- Faction tree - Voracity (consumables): Candy effect gives up to +50% happy from candy; Energy drink effect up to +50% energy from energy drinks; Alcohol effect up to +50% nerve from alcohol; Booster cooldown adds up to +24 hours of max booster cooldown. [source: Torn API (faction tree) | 2026-07-17]
- Faction tree - Toleration (drugs): Side effects reduces drug side effects and boosts addiction mitigation up to 30%; Overdosing reduces overdose risk up to -30%; Addiction I-XXV reduces drug addiction gain up to -50%. [source: Torn API (faction tree) | 2026-07-17]
- Faction tree - Excursion (travel/money): Hunting gives up to +30% hunting income; Offshore banking up to +25% Cayman bank interest; Travel capacity adds up to +10 travel items; Travel cost cuts travel fees up to -75%; Rehab cost reduces rehab cost up to -20%. [source: Torn API (faction tree) | 2026-07-17]
- Faction tree - Steadfast (gym gains): Strength, Speed, Defense, and Dexterity training branches each add up to +20% gym gains for that stat, gated behind energy spent training that stat. [source: Torn API (faction tree) | 2026-07-17]
- Faction tree - Aggression (offense): Hospitalization adds up to +50% outgoing hospital time; Damage I-X gives up to +10% all damage; Strength and Speed branches each give up to +20% passive; Accuracy I-X gives up to +2.0 all accuracy. [source: Torn API (faction tree) | 2026-07-17]
- Faction tree - Suppression (defense): Maximum life gives up to +20% max life; Defense and Dexterity branches each give up to +20% passive; Escape I-X boosts dexterity by up to +500% during an escape attempt. [source: Torn API (faction tree) | 2026-07-17]
