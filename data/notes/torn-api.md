- Torn API (overview): the API is Torn's official, fully-supported, READ-ONLY tool for pulling data about your player, faction, or company using a 16-character API key. You can build extensions, apps, or websites with it. Start at api.torn.com (click "try it"); your API key is under the "API key" section of your Preferences page. To list available selections for a type, call e.g. http://api.torn.com/user/?selections=lookup&key=YOURKEY. [source: https://wiki.torn.com/wiki/API | 2026-07-18]
- Torn API - keys & access levels: a user can hold 25 keys, each separately deletable, with 4 access levels - "Public Only", "Minimal Access", "Limited Access", and "Full Access" - each restricting what data the key can fetch. Apps should only ever ask for the key, never a name, user ID, or (absolutely never) your Torn password. Keep other people's keys and their data confidential; misused apps get permanently banned. [source: https://wiki.torn.com/wiki/API | 2026-07-18]
- Torn API - rate limits: each user can make up to 100 requests per minute - hitting the limit temporarily bans the key (error 5). Repeated requests with invalid keys can cause a temporary IP ban (error 8), so scripts must drop disabled/invalid keys on error. Scripts should request only the data they need to reduce server load. Limits can change without notice. [source: https://wiki.torn.com/wiki/API | 2026-07-18]
- Torn API - error codes: 0 unknown, 1 key empty, 2 incorrect key, 3 wrong type, 4 wrong fields, 5 too many requests (>100/min, key banned briefly), 6 incorrect ID, 7 incorrect ID-entity relation (data is private), 8 IP block (abuse), 9 API disabled, 10 key owner in federal jail, 11 key change error (once per 60s), 12 key read error, 13 key disabled from owner inactivity (inactive 7+ days), 14 daily read limit reached, 15 temporary/test error, 16 access level too low, 17 backend error (retry), 18 key paused by owner. [source: https://wiki.torn.com/wiki/API | 2026-07-18]
- Torn API - known safe applications: well-known trusted apps that use your API key include Torn Stats (by IceBlueFire), TornTools (by Mephiles), YATA (by Kivou), and Torn PDA (by Manuito). [source: https://wiki.torn.com/wiki/API | 2026-07-18]
